### 文体层（风格锚点）

**基础叙述温度**：冷叙述 + 克制抒情
- 写灾难像写天气预报，不携带情感判断
- 悲伤靠外部细节暗示（"落得比旗帜还慢"），不直接写内心
- 最狂暴的场面（古筝行动）用最精确冷静的语言

**温度变化规则**：
- 科学理论段 → 纯冷叙述，接近零度书写
- 三体游戏世界 → 冷叙述为主，宏大场面时微微升温
- 叶文洁往事 → 克制抒情，情绪被压住靠细节暗示
- 古筝行动 → 极端冷静写极端暴烈
- 结尾"虫子" → 暗涌型，平静语句下蓄积力量

**对白风格**：
- 基础：偏书面，知识分子科学对话
- 史强例外：全口语化、短句、锋利
- 三体游戏角色：戏剧化，历史人物说现代话
- 潜台词密度：中高，科学讨论包裹文明追问

**节奏型**：交替型——宏大场景快节奏 + 人物沉浸慢节奏交替进行

**原作禁区**：不煽情、不进行道德评价、心理活动只通过外部行为暗示

## 三、改编方向

- **玩家视角**：扮演汪淼，第二人称"你"——从科学边界调查者到真相承载者
- **叶文洁往事**：通过对白与发现间接呈现，玩家作为"倾听者"
- **三体游戏**：保留全貌，每个文明纪元是一个独立场景群
- **核心张力**：一个绝望的文明能否救赎另一个绝望的文明？
- **改编基调**：冷叙述主调，科学对话原作风味，史强全口语化

## 四、分支拓扑设计

### 四幕结构

```
第一幕：疯狂年代  ──→  第二幕：科学边界  ──→  第三幕：真相浮现  ──→  第四幕：黑暗降临
  [叶文洁往事]        [汪淼调查线]           [两线交汇]              [结局分叉]
      |                     |                    |                     |
  ①批斗会             ⑧被约谈             ⑭叶文洁讲述          ⑲物理学崩塌
  ②到大兴安岭         ⑨三体游戏Ⅰ         ⑮ETO集会             ⑳虫子对话
  ③红岸秘密          ⑩三体游戏Ⅱ         ⑯古筝谋划            ㉑→ 结局
  ④发信               ⑪三体游戏Ⅲ         ⑰古筝执行
  ⑤三体回应          ⑫宇宙闪烁           ⑱智子真相
  ⑥杀雷/杨           ⑬三体游戏Ⅳ
  ⑦遇伊文斯
```

### 分支类型分布（18个分支点）

| 类型 | 数量 | 占比 | 说明 |
|------|:----:|:----:|------|
| 态度分支 | 12 | 67% | 不改变情节，改变余烬值/关系值/flag |
| 路径分支 | 3 | 17% | 经历不同场景，最终汇合 |
| 命运分支 | 3 | 17% | 影响结局走向的关键选择 |

### 汇合点（4个漏斗）

| 汇合点 | 位置 | 设计 |
|--------|------|------|
| 汇合① | 第一幕末 | 无论玩家对叶文洁的态度如何，都进入"现在→汪淼"的转场 |
| 汇合② | 第二幕末 | 不同探索深度的玩家都确认"三体游戏=真实文明" |
| 汇合③ | 第三幕中 | 古筝行动前所有路径归一，共享核心事件 |
| 汇合④ | 第三幕末 | 全部知晓智子真相 |

### 分支文学大纲（节选）

**主线路径（对应原作·「虫子」结局）**：
> 玩家按汪淼的轨迹——从被约谈时的困惑，到三体游戏中的震撼探索，到倾听叶文洁往事时的复杂感受，经历古筝行动的屏息时刻，最终站在物理学崩塌的废墟上，在史强"虫子"宣言中找到某种坚定。这条路径上，玩家虽然经历了绝望，但选择相信"虫子从来没有被战胜过"——不是因为乐观，而是因为这是唯一能做的事。

**加入ETO线（偏离原作·「救世主幻觉」结局）**：
> 分叉点在第二幕末至第三幕初。如果玩家在各分支选择中对人类文明持续表现出失望和质疑（余烬值走低），同时在ETO集会场景中选择"理解他们"，路径将偏离原作。玩家被伊文斯说服，相信三体文明能带来更高等的秩序。这条线通向的不是胜利——而是在三体信息被截获后，发现三体文明的真相远比想象中冷酷，自己被利用而不自知。

**逃避线（「漫长的睡眠」结局）**：
> 分叉在第四幕初。玩家在知晓全部真相后，选择不面对。通过汪淼的关系申请冬眠，计划在四个半世纪后三体舰队到达时再醒来。这条路径的结尾是：冬眠醒来，被告知地球已被占领，而自己被作为"历史遗物"保留——漫长睡眠后的更漫长噩梦。

## 五、状态系统

### 主状态值：余烬（0-100）

三体1最核心的心理张力——**对文明的信心**。起始 35。

命名取自小说的核心意象：文革的灰烬→红岸的余火→虫子的残光。

| 范围 | 状态 | 叙事表现 |
|:----:|:----:|---------|
| ≥70 | 灼热 | 对人类文明坚定信任，选择主动面对 |
| 40-69 | 余温 | 摇摆，既不完全相信也不彻底绝望 |
| ≤30 | 灰烬 | 对人类文明失去信心，倾向逃避或顺从 |

### 自定义变量

| 变量 | 类型 | 初始 | 说明 |
|------|:----:|:----:|------|
| `wenjie_trust` | int 0-10 | 5 | 对叶文洁的信任度，影响与她对话时的选项 |
| `game_depth` | int 0-10 | 0 | 三体游戏探索深度，影响可解锁的内容 |
| `knows_truth` | bool | false | 是否知晓三体文明的完整真相 |
| `eto_stance` | str | "对抗" | 对ETO的立场："对抗"/"理解"/"加入" |
| `shi_qiang_bond` | int 0-10 | 3 | 与史强的默契度，影响互动质量 |

## 六、结局矩阵

| 结局 | 类型 | 到达条件 | 一句话 |
|:----:|:----:|---------|--------|
| **虫子** | TRUE ENDING | 余烬≥60 + `decided_fight` | "虫子从来没有被战胜过。" |
| **救世主幻觉** | NORMAL ENDING | 余烬≤30 + `eto_stance`="加入" + `attended_eto` | 加入ETO，最终发现被利用。 |
| **漫长的睡眠** | NORMAL ENDING | 余烬≤40 + `decided_flee` + `knows_truth` | 逃避一切，在冬眠中错过所有。 |
| **深渊回响** | BRANCH ENDING | 余烬≤20 + `eto_stance`="加入" + `knows_sophon` | 主动协助三体，成为文明叛徒。 |
| **最后的切割** | RASH ENDING | 古筝行动中选择自我牺牲 | 用生命确保信息被截获。 |
| **无尽游戏** | HIDDEN ENDING | `game_depth`≥8 + `!decided_fight` + `!decided_flee` + `!decided_join_eto` | 永远留在三体游戏的世界里。 |

### 兜底规则

- 任意路径（无论怎么选择）都能到达至少一个结局
- 余烬值的极端情况（≥90 或 ≤10）会屏蔽中间度结局、强制指向两极
- 未触发任何关键 flag 的情况下，默认导向「漫长的睡眠」
- 无死胡同：每个非结局节点都有 choices/next/routes

## 九、质量基线

| 维度 | 标准 | 验证方法 |
|------|------|---------|
| 节点可达性 | 所有节点从startNodeId可达 | `validate.py` BFS遍历 |
| 引用完整性 | 所有next指向存在的节点ID | `validate.py` 检查 |
| 无死胡同 | 非ending节点均有出路 | `validate.py` 检查 |
| 结局可达 | 每条路径至少一个结局 | `validate.py` 检查 |
| progress单调 | 路径上progress大致递增 | 手动抽查 |
| 成就数 | > 结局数 | 8 > 6 ✓ |
| 反压缩 | 情节/对白/过场不省略 | 写作时逐章自检 |
| 回响 | 每个选择有即时+持续+结局三层影响 | 写作时逐节点检查 |
