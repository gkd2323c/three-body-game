## 一、双轴属性系统

第三部采用双轴属性取代单一 val，以精确映射程心在不同维度上的心理状态。

### 轴A（主显示值 val）：人性

**variableName：人性**

**含义：你选择成为一个什么样的人——在宇宙的冰冷法则面前。**

> "失去人性，失去很多；失去兽性，失去一切。" ——维德

高「人性」意味着：**温暖、慈悲、母性本能、以爱行事——但在黑暗森林中，爱可能致命。**
低「人性」意味着：**冷酷、理性、不择手段、生存优先——可能拯救文明，但代价是失去自我。**

| 范围 | 状态描述 | 体验质感 |
|:----:|:---------|:---------|
| 0-20 | 绝对零度——完全理性 | 维德式冷酷。每个选择都是精确计算。失去人性但可能拯救文明 |
| 21-40 | 冰点——克制的温度 | 罗辑式平衡。知道什么时候该放下感情 |
| 41-60 | 体温——正常的人类温度 | 程心初始状态。善良但不极端 |
| 61-80 | 暖流——温情主导 | 母性本能上升。保护身边人的冲动超过战略考量 |
| 81-100 | 燃烧——爱到毁灭 | "以爱的名义把世界推向深渊"。程心式悲剧的极端表现 |

**initialVariable：50**

### 轴B（隐性变量）：威慑度

**变量名：deterrence（威慑度）**

**含义：三体世界对你作为执剑人的战略评估（0-100）。仅在第2-3章活跃显示。**

> "你的威慑度是百分之十。罗辑是百分之九十。维德是百分之一百。" ——智子

| 范围 | 三体评估 | 战略后果 |
|:----:|:--------:|:---------|
| 0-15 | 无威胁 | 三体立即发动攻击。威慑必然失败 |
| 16-40 | 低威胁 | 三体大概率攻击，但会保留观察窗口 |
| 41-60 | 中等威胁 | 三体犹豫，可能试探性攻击 |
| 61-85 | 高威慑 | 三体不敢轻举妄动。威慑维持 |
| 86-100 | 绝对威慑 | 三体完全不敢行动。罗辑/维德级别 |

**威慑度的计算规则：**

| 行为 | 威慑度变化 | 逻辑 |
|:-----|:----------:|:-----|
| 选择冷酷/理性选项 | +5~+10 | 三体通过智子评估你的行为模式 |
| 选择温暖/慈悲选项 | -5~-10 | 母性本能被三体解读为"不会按下按钮" |
| 维德伏击后选择愤怒回应 | +8 | 三体看到你的攻击性 |
| 维德伏击后选择原谅 | -12 | 三体确认你"太善良" |
| 接受候选人劝告不参选 | N/A（维德接手，威慑度自动锁定95+） | 三体面对维德 |
| 执剑人交接时表现出犹豫 | -15 | 三体精准捕捉 |

**威慑度在第1章和第4-5章不活跃**（不存在执剑博弈），但人性轴的选择会间接影响后续路径。

### 双轴组合效应

| 人性 | 威慑度 | 组合效果 |
|:----:|:------:|:---------|
| 低 | 高 | **维德模式**——冷酷且有威慑力。三体不敢动。最佳生存路线 |
| 低 | 低 | **失败的冷酷**——想理性但三体不买账。危险 |
| 高 | 高 | **罗辑悖论**——善良但三体不敢赌。罕见组合（需要特殊路径） |
| 高 | 低 | **程心模式**——温暖但无威慑力。威慑必然失败。原作路径 |

### 两轴的叙事效果

**人性轴影响（全章节）：**
- **NPC 态度**：高人性时智子更温柔/关怀；低人性时智子更直接/务实
- **可获取信息**：低人性时维德更愿意透露计划；高人性时他保持沉默
- **segments 文本色调**：高人性时环境描写更暖、更柔和；低人性时更冷、更空旷
- **云天明通话情感深度**：高人性时通话情感更浓烈；低人性时通话更侧重情报传递

**威慑度影响（仅第2-3章）：**
- **三体行为模式**：高威慑度时三体更谨慎/尊重；低威慑度时三体更轻蔑/激进
- **智子台词差异**：威慑度≥60时智子用敬语；<30时智子用蔑称（"可怜虫"）
- **假警报事件的惨烈度**：高威慑度→警报被更快控制；低威慑度→踩踏和恐慌加剧
- **选项可用性**：威慑度≥70时出现"以威慑换和平"的选项；<30时此选项消失

| 曲率驱动顿悟 | 程心叠纸船实验成功 | +25 |

| 安全声明 | 完整逻辑链 | +15 |

| 解读耐心加分 | 在IDC阶段未跳过任何解读环节 | +10 |

**clue_score 阈值效果：**

| 阈值 | 效果 |
|:----:|:------|
| ≥80 | **完全理解**——解锁隐藏路线：提前向星环集团披露曲率驱动原理，在假警报前暗中布局光速飞船，降低后续维德对抗联邦的惨烈度。冥王星上程心完全理解光速飞船=安全声明悖论 |
| 40-79 | **部分理解**——标准路径。程心知道情报重要但不完全理解后果。冥王星上的觉悟有延迟但最终到达 |
| <40 | **理解不足**——必须消耗游戏内时间请曹彬协助解读，后续战略窗口缩短。冥王星上程心错过最佳选择时机，某些高阶选项不可用 |

### 巨型分支处理规则（Major Divergence Protocol）

当分支选择导致后续大量节点逻辑不成立时，采用以下协议：

**分支点7（程心退选）的处理：**

如果玩家选择 A（听取劝告不参选），后续 C47-C51（程心执剑人交接、水滴攻击、扔掉开关）在逻辑上不成立，因为执剑人不是程心。

**采用「快速收束」方案：**

```
选择A后 → C43a [对白] 程心宣布退选，PDC和民众震惊
         → C43b [场景] 媒体风暴。"圣母抛弃人类"的标题铺天盖地
         → C43c [对白] 维德竞选获胜。但三体评估：维德威慑度98%
         → C43d [信息] 时间跳跃：五年后。威慑维持。三体不敢动
         → C43e [场景] 程心在地球过着平凡生活。但窗外的世界在变
         → C43f [信息] 曲率航迹被发现。光速飞船被禁。程心预感不对
         → 快速收束到 C89（维德回归）——此时维德掌控星环公司
```

快速收束节点数：6-10个（远少于主线的20+个），但完整覆盖了"威慑维持"的世界状态变化。最终导向 BRANCH ENDING 4（另一个执剑人）。

**分支点12（程心支持维德）的处理：**

如果玩家选择 B（支持维德），C99-C102（维德被处死、游荡地球）不成立。

**采用「镜像平移」方案：**

```
选择B后 → C98a [对白] 程心："前进。"维德眼中闪过罕见的光
         → C98b [信息] 时间跳跃：30年。曲率驱动取得突破
         → C98c [场景] 第一艘光速飞船原型在星环城完成
         → C98d [信息] 二向箔到达前5年。人类已有3艘光速飞船
         → 镜像平移至 C107（二向箔到来），但此时人类有逃亡能力
         → 导向 BRANCH ENDING 5（星舰地球）
```

**核心原则：巨型分支必须完整覆盖时间线，不能留下逻辑真空。**

### META ENDING：漂流瓶

**核心设定**：玩家经历的所有结局——真结局、草率结局、分支结局——并非"读档重来"的游戏机制，而是647号小宇宙中神经元计算机在1890万年间对太阳系文明进行的量子概率推演。每一个结局都是概率空间中一条被完整记录的"文明演化切片"。

#### 隐性框架：量子毛刺

在多次游玩过程中，某些关键节点的描述文本偶尔出现"量子毛刺"——0.5秒后自动消失的概率重叠记录。毛刺文本遵循风格锚点的"纯科学温度"规则：冰冷、精确、无主观抒情，模拟概率分支的叠加态坍缩。

**触发节点与毛刺文案：**

**C49（执剑人交接时刻）**
```
[标准] 你把红色开关握在手中。铜质边缘传来罗辑的体温。
[第N次] 你把红色开关握在手中。铜质边缘传来罗辑的体温。
[量子毛刺·0.5秒] 发射台在屏幕上化为一片二维的雪花。不，它仍是立体的。你正握着开关。
```

**C98（星环城对峙时刻）**

## 六、meta 配置
