## 一、叙述温度

**主调：冷叙述中的炽热内核**

刘慈欣的叙述温度是"冰下有火"——外层极度克制、科学化、零度情感，但关键瞬间爆发出压倒性的崇高感。

### 规则

1. **日常叙述温度：零度偏冷**
   - 不抒情、不感叹、不煽情。用精确的数据、物理描述、空间坐标代替情感形容词
   - ✅ "辐射帆短暂反射阳光到北半球，像一只在太空中看了一眼地球后慢慢闭上的眼睛"
   - ❌ "壮丽的辐射帆令人热泪盈眶，仿佛人类最后的希望在太空中闪耀"

2. **情感爆发时刻：允许升温但必须通过具象**
   - 只在极少数时刻允许"热"叙述，且必须通过具体意象而非抽象情感词
   - ✅ "程心带来气流冲散白雾——短暂的寒气拥抱后消散"
   - ❌ "程心感到无尽的悲伤和绝望，泪水夺眶而出"

3. **科幻概念叙述：纯科学温度**
   - 涉及物理概念时，温度降至绝对零度。用类比和数字而非情感
   - ✅ "三维跌落二维如四维跌落三维——维度打击的物理本质"
   - ❌ "令人恐惧的维度打击让所有人都感到绝望"

4. **宇宙景观叙述：崇高而非恐怖**
   - 描写宇宙尺度事件时，保持敬畏感而非恐惧感。壮美优先于恐怖
   - ✅ "太阳的血铺展在二维平面中，这是最后一次日落"
   - ❌ "恐怖的二维化吞噬了太阳，所有人都吓得魂飞魄散"

## 二、对白风格

### 程心
- **风格**：礼貌、真诚、偶尔怯生生但关键时刻有力量。自省式内心独白
- **句长**：中等偏长，复合句。常带转折和自问
- **特征**：不说空话，每句话都有具体指涉。自我道德审判常嵌入叙述
- **示例**："他吃什么？！" / "如果所有小宇宙中的人都这么想，那大宇宙肯定死了"

### 云天明
- **风格**：极度寡言。只在极端时刻爆发。内心世界极其丰富但表达极度匮乏
- **句长**：极短。偶尔的长句只出现在回忆或独白中
- **特征**：引用卡夫卡。沉默是常态，不沉默是事件
- **示例**："在这个世界里我感到自己是个外人" / "我们的星星"

### 维德
- **风格**：极简、冰冷、精准。每句话如"一个致密的线团"
- **句长**：极短。命令句为主。偶有黑色幽默
- **特征**：面无表情如雕像。不解释，不安慰，不道歉
- **示例**："不择手段地前进" / "失去人性，失去很多；失去兽性，失去一切" / "谢谢你的雪茄"

### 艾AA
- **风格**：大嗓门、直爽、情绪外放。用现代人眼光解构一切
- **句长**：短到中等。感叹句多
- **特征**：生活化口语。偶尔粗口。是全书的"人间烟火"
- **示例**："你傻不傻呀？！" / "这个混蛋、恶魔、杀人犯……他行。"

### 智子
- **风格**：两个面——温柔面（茶道，轻言慢语）和暴力面（迷彩，简短利落）
- **句长**：温柔面中等偏长；暴力面极短
- **特征**：极度克制的文明人。在毁灭面前展现"人类无法企及的高贵和尊严"
- **示例**："宇宙很大，生活更大" / "可怜虫们，准备去澳大利亚吧！" / "排队。"

### 罗辑
- **风格**：威慑纪元全程无言。冥王星时期苍老、平静、超越生死
- **句长**：极短。最后的话如遗言
- **特征**：不解释、不后悔。用沉默和行动说话
- **示例**："哦，要进画里了，孩子们，走好。" / "一切伟大的建筑都埋在黑影里"

### 关一帆
- **风格**：学者式激情+深沉恐惧。时而疯狂时而沉默
- **句长**：中等偏长。描述宇宙时常有长句
- **特征**：将恐惧转化为学术语言。"暗无天日"是口头禅
- **示例**："方寸之间，深不见底" / "这一切，暗无天日" / "你没有错"

### 歌者
- **风格**：日常工作式随意。毁灭星系如打扫卫生
- **句长**：短。碎片式思维
- **特征**：吟唱古歌谣。直觉判断。无情感波动
- **示例**："有诚意" / "给，给你"（长老给二向箔）

### 选项文本（玩家可点击的选择）

选项是程心的内心映射，不是"玩家指令"。每个选项必须读起来像是此刻这个角色脑中真正闪过的一个念头或一个可能的行动。

**三条铁律：**

1. **必须是程心的声音**——充满自省、带有道德两难、或是客观克制的行动指令。不允许跳出角色的上帝视角、旁白式评论、或轻浮口吻
2. **信息时序约束**——选项中不能出现玩家此刻尚未获知的人名/地名/概念（与第一部、第二部同一铁律）
3. **情境匹配**——选项只回应此刻正在发生的事，不预知后续剧情

**风格分类：**

| 类型 | 适用场景 | 语气 | 示例 |
|:----:|:--------:|:-----|:-----|
| 行动指令 | 明确的物理动作 | 克制、简洁 | "把红色开关握在手中" / "向灰星飞去" |
| 内心独白 | 道德两难时刻 | 自省、犹豫 | "如果这就是正确的选择，为什么我的手在发抖？" / "他不该被这样对待" |
| 对白台词 | 说话选项 | 口语化、角色化 | "不。停止抵抗。" / "走吧，我来处理" |
| 认知选择 | 信息理解分岔 | 理性、克制 | "关注故事本身——那些隐喻在说什么" / "看着他的脸——三个世纪的守望" |

**反面教材（W19 执剑人交接时刻）：**

```
❌ "跟三体人拼了，老子就是不放手！"     → 轻浮、非程心语气、现代网络烂梗
❌ "启动引力波广播，毁灭三体！"          → 过于直白，缺少内心挣扎
❌ "我选择保护人类！"                    → 口号式，不是真实想法

✅ "把红色开关握在手中。感受铜质边缘传来的微微体温。"   → 行动+感官，克制
✅ "闭上眼睛。在这个位置上，罗辑坐了五十四年。"         → 内心独白+历史感
✅ "扔掉它。像扔掉一块烧红的铁。"                       → 行动+比喻，程心式决断
```

**反面教材（第4章星环城对峙）：**

```
❌ "打就打，谁怕谁！"                   → 轻浮，非程心
❌ "我支持维德，人类需要光速飞船"        → 旁白评论，不是选项

✅ "看着那条金色弹链。每颗子弹都装着足以摧毁一座城市的反物质。"
✅ "必须阻止他。无论代价是什么。"
```

## 三、节奏型

**原作节奏型：交替型（短句急促+长句绵延交替）**

### 规则

1. **场景建制段**：长句绵延，建立空间感和时间感。允许3-5句不给互动
2. **冲突/转折段**：短句急促。每1-2句给一次互动。句子如鼓点
3. **信息交付段**：中等句长，数据密集。类比+数字交替。每3-4句给互动
4. **情感高潮段**：长短交替。短句制造冲击，长句铺展感受。允许5-8句不给互动（结局/高潮特权）
5. **哲学/宇宙观段**：长句为主，层层递进。允许4-6句不给互动

### 互动节奏硬性规则

**每 1-5 句正文必须给玩家一次互动。** 长段落只允许出现在：结局、独白高潮、真相揭露、宇宙景观描写。

## 四、视角与人称

### 原作视角
- 原作第三人称全知视角，但紧跟程心的感知
- 云天明的内心独白（第一部）以独立章节呈现
- "万有引力"号线以关一帆视角为主
- 歌者段落以歌者视角

### 游戏视角转换规则
- **全程第二人称"你"**（与前两部一致）
- 云天明的第一部章节：改为"你回忆起……"（程心通过后续信息获知云天明经历）
- "万有引力"号线：通过程心的冬眠"错过"来处理——她苏醒后才得知这些事件
- 歌者段落：通过程心的"想象"或信息碎片间接呈现（不在玩家视角中直接出现歌者）

### 信息隔离规则
- 程心在每个时间点只知道她能知道的事
- 云天明送星者的身份、维德的真实计划、罗辑的秘密等，必须等到原作揭示时刻才能让玩家获知
- "万有引力"号线的事件在程心苏醒后通过他人叙述/记录呈现

## 五、原作禁区

### 绝对禁止

1. **禁止心理描写替代行动**——不用"她意识到""她突然明白"来交代顿悟，必须通过具体行动/对话/发现来展示
2. **禁止说教和催泪**——不用"这让人深思""多么伟大的母爱"等评价性语句
3. **禁止HTML标签**——正文纯文本
4. **禁止元叙事词汇**——"玩家""节点""分支""选项"等绝对不能出现在正文和选项中
5. **禁止压缩信息**——任何承载角色性格、情节推进或氛围建立的段落不允许省略

### 刘慈欣的特殊禁区（风格锚点）

6. **不用成语堆砌**——原作极少用四字成语，多用创造性的比喻
7. **不用感叹号收束情感段**——原作的情感高潮多用句号或省略号，感叹号只出现在角色台词中
8. **不解释科学概念的情感意义**——只描述物理本身，让读者自己感受震撼
9. **不用"仿佛""好像""如同"超过一次/段**——原作比喻精确但节制
10. **不用"突然""忽然"开场转折**——用具体的时间/动作触发转折

### 情感关键词
- 温差（人性温度vs宇宙冰冷）

- 选择（不可逆的、以爱的名义）
- 守望（跨光年、跨世纪）

- ❌ 美丽、壮丽、震撼（用具体描写代替形容词）
- ❌ 终于、突然、忽然（用具体触发事件代替）
