# 《三体》互动文游改编方案 · 修订补充

> 基于第一次审查反馈的修订方案。
> 替代原方案中的对应章节，未覆盖的部分以原方案为准。

---

## 修订1：体量压缩（替代原·输出分批计划）

**新的目标：5000-6000 节点，40-50 万写作量**

| 调整项 | 原方案 | 修订后 | 理由 |
|--------|:------:|:------:|------|
| 总节点数 | 7000-8000 | **5000-6000** | 压缩非关键分支节点密度 |
| 总批次数 | 15 批 | **10-12 批** | 每批可扩展到 100-120 节点 |
| 每千字节点 | 35-40 | **25-30** | 放宽容积，减少非关键过场 |
| TRUE ENDING 节点 | ~1500 | **~2500** | 优先做主线完整链路 |
| 分支结局总节点 | ~5500 | **~2500-3500** | 先做核心线，后置简化 |

**执行策略**：
- TRUE ENDING 路径（主线路径）优先完成全节点链，第一批开工
- 分支结局线先完成核心决策节点 + 结局节点，中间过渡段可缩略
- 第一批试写后校准实际密度

## 修订2：三体游戏交互深度（替代原·分支拓扑对应章节）

**设计原则：线性推进 + 关键选择，取消网格式探索**

| 文明纪元 | 处理方式 | 分支设计 |
|---------|---------|---------|
| 周文王/纣王（139号文明） | **线性 + 1 个态度选择** | 作为三体游戏的首次体验，保持线性沉浸。唯一选择：面对纣王的残暴——恐惧/冷静/好奇 |
| 墨子 | **线性 + 1 个态度选择** | 玩家在墨子展示其宇宙模型后，选择：相信/质疑/追问 |
| 哥白尼/三日凌空 | **线性** | 纯推进，无分支 |
| 人列计算机（秦始皇） | **线性 + 1 个路径选择** | 冯·诺依曼邀请观看人列计算机运行——观看/帮助设计/去找其他学者 |
| 爱因斯坦/大撕裂 | **线性 + 1 个态度选择** | 面对三体文明的真相——震撼/接受/怀疑 |

game_depth 作用：从「解锁更多文明」改为「解锁隐藏信息和对话选项」。
- game_depth≥3：在叶文洁对话中解锁更深层的追问
- game_depth≥5：解锁三体文明的额外背景信息
- game_depth≥8：通向隐藏结局的必要条件

**效果**：三体游戏部分从 8-10 个分支点缩减到 5 个关键选择，每个选择承载明确的叙事功能。

## 修订3：叶文洁往事互动设计

在叶文洁讲述过程中嵌入**微互动**，避免"看播片"感：

| 触发位置 | 互动形式 | 影响 |
|---------|---------|------|
| 叶文洁讲到父亲被打死时 | 沉默/追问细节/转移话题 | wenjie_trust ±1-2 |
| 叶文洁讲到大兴安岭时 | 选择关注环境/关注人心/关注技术 | 余烬 ±3 |
| 叶文洁讲到发送信号时 | 认同她的决定/质疑/沉默 | wenjie_trust ±2，余烬 ±5 |
| 叶文洁讲到杀死雷/杨时 | 震惊/理解/追问动机 | wenjie_trust ±3 |
| 讲述过程中 | 可插入「跳过细节，直接听结论」 | 一次跳过机会 |

**跳过机制**：每个回忆段如果玩家选择"跳过"，以 2-3 句摘要告知核心信息，然后进入下一段。跳过的代价：错过细节 → 影响最终 wenjie_trust 判断的精度。

**保障**：主干剧情（红岸发信→三体回应→死亡事件）不可跳过，可跳过的是情绪铺陈段。

## 修订4：分支结局宽松判定

| 结局 | 原条件 | 修订后 | 宽松理由 |
|:----:|--------|--------|---------|
| 救世主幻觉 | 余烬≤30 + eto_stance="加入" | **余烬≤40 + 累计3次"对人类失望"选择** | 不依赖显式 eto_stance 赋值，行为累积判定 |
| 漫长的睡眠 | 余烬≤40 + decided_flee | **余烬≤50 + 累计2次回避选择** | 降低阈值，更易触发 |
| 深渊回响 | 余烬≤20 + eto_stance="加入" | **余烬≤25 + 累计4次"对人类失望" + 主动协助ETO** | 提高门槛确保隐藏度 |
| 最后的切割 | 古筝行动中选择牺牲 | **保留，增加余烬≥70激活** | 高余烬值玩家才可能触发此线 |
| 无尽游戏 | game_depth≥8 + 特殊条件 | **保留，增加一个条件：未选择任何结局导向行为** | — |

**新增宽松判定记录变量**：
- `despair_count`: int，记录玩家表达"对人类失望"的累计次数
- `avoid_count`: int，记录玩家选择"逃避"的累计次数
- 当上述变量达到阈值时，即使未选显式分支也可触发对应结局

### 变化幅度设计原则

- **重大事件**（智子真相、虫子宣言）：±10-15，单次足以显著改变走向
- **日常选择**（态度变化、追问）：±2-5，累积效应
- **累计变化控制**：正常情况下玩家会经历约 10-15 次取值变化，全程可积累 ±50-80 的变动空间，确保从起始 35 出发可到达两端

## 修订6：分支选择设计规范（新增）

### 选项类型与字数限制

| 类型 | 字数 | 示例 |
|------|:----:|------|
| 态度选项 | 2-6 字 | "继续听" / "追问细节" / "沉默" |
| 行动选项 | 4-10 字 | "走向窗前" / "翻开那封信" |
| 对白选项（台词） | 带引号，3-15 字 | '"这是真的吗？"' / '"我不相信。"' |
| 单选项确认 | 2-6 字 | "继续" / "走过去" |

### 选项设计纪律

1. **情境匹配**：选项只回应此刻正在发生的事，不预知后续
2. **信息时序**：选项中出现的人名/概念必须是玩家在此路径上已经遇到过的
3. **禁止「说XXX」格式**：说话直接写台词加引号，动作直接写动作
4. **每个选项必须有回响**：至少即时反应（下一屏文字因选择而不同）
5. **态度选项不用「是/否」**：用情境化的两个不同表达

### 分支密度控制

| 场景类型 | 选择间隔 | 示例 |
|---------|:--------:|------|
| 日常/探索 | 每 3-5 句一次 | 在三体游戏中观察描述后给出选择 |
| 紧张/行动 | 每 1-3 句一次 | 古筝行动中的快速抉择 |
| 沉浸/回忆 | 每 5-8 句一次 | 叶文洁讲述中的微互动 |
| 结局前收束 | 每 2-3 句一次 | 结局分叉前的密集选择 |

### 状态变更纪律

- 每次选择最多变更 2 个状态值（余烬 + 1 个变量/flag）
- 结局分叉点除外（可同时变更 3 个）
- changes 必须在选择后下一屏有对应的叙事反馈

### 新增隐藏结局「破壁人」

条件：在 ETO 集会场景中选择"深入追问降临派的真实目的" + 古筝行动中选择亲自查看三体信息 + 后续解读时选择"发现矛盾，三体文明的叙述有缺口"。

信息边界说明：叶文洁讲述阶段玩家无法追问"三体文明的真实意图"（她当时不知道），改为追问"三体世界为什么只发来'不要回答'"——这个疑问在叶文洁所知范围内是合理的。真正的破壁线索要到 ETO 集会和截获三体信息后才出现。

内容：玩家在截获的三体信息中发现了一条只有自己看懂的暗示——三体文明对地球的叙述存在一个根本性矛盾。玩家意识到自己可能是唯一看出问题的人，成为人类文明的"破壁人"，独自探索真相。
