# 风格锚点：《三体1：地球往事》

## 一、叙述温度

### 基础温度：冷叙述 + 克制抒情交替

《三体1》的风格与《三体2》不同——它不是单一的冷叙述，而是在**冷叙述（科学线）** 和**克制抒情（历史线）** 之间交替：

| 段落类型 | 基础温度 | 特征 |
|---------|---------|:-----|
| 现代科学线（汪淼/丁仪/作战中心） | **冷叙述** | 不带情感评价，记录台球实验结果、倒计时数字、宇宙辐射数据 |
| 历史记忆线（叶文洁/红岸） | **克制抒情** | 有情感但被压制，靠外部细节暗示（夕阳、雪、杨冬的画） |
| 三体游戏线 | **神话/寓言式叙述** | 周文王、纣王、脱水浸泡——用古代叙事框架包裹科幻设定 |
| 史强线 | **反讽** | 粗俗的外壳下藏着锋利的洞察，用"邪招"解构一切庄严 |
| 高潮段（古筝行动） | **极冷** | 用精确的技术细节写一场大屠杀——"飞刃"切割成为纯粹物理现象 |

### 温度变化规则

```
科学调查段：冷叙述，只写看到的数据和现象
 → 三体游戏段：温度略升，神话寓言色彩
 → 叶文洁回忆段：克制抒情，夕阳/雪/血迹的意象反复出现
 → ETO集会段：回到冷叙述，记录一场狂热的政治集会
 → 古筝行动段：极冷——最残忍的场景用最冷静的笔触
 → "虫子"段：克制抒情→升华，不煽情但有力
```

### 游戏正文指引
- **科学线**segments 默认用短句、不带修饰的叙述，只写观察到的现象
- **历史线**允许在关键情绪节点出现感官细节（气味、温度、光线），但不出现直接情绪形容词
- **三体游戏线**使用略微古朴的措辞，对白偏文言风格（周文王/纣王）

## 二、对白规则

### 通用规则
- 科学家对白偏书面精确，尽量省略语气词
- 史强全口语化，这是全书唯一的"出口"
- 对白密度高——信息主要通过对话传递
- 潜台词：叶文洁的对白潜台词密度最大（说的和想的不一致）

### 角色对白规则

#### 汪淼
```
句长：中短句为主
口语程度：中——学者口吻，偏书面但不端着
特征：遇到反常现象时先分析后表达情绪
核心句式：喜欢用问句（"这意味着什么？"  "这怎么可能？"）
```

#### 史强
```
句长：短句，急促
口语程度：极高——全书最口语化的角色
口头禅："扯淡"、"他妈的"、"熊样儿"
标志：喜欢用市井比喻（"肉包子打狗"、"像洗面口袋似的"）
潜台词：几乎没有——说什么就是什么，但粗俗里藏着精准
```

#### 叶文洁
```
句长：中等偏短，但每个字密度极大
口语程度：中——学者的简练
特征：不解释多余一句。面对汪淼时温和克制，面对审问者时冷静精准
关键场景：尾声"这是人类的落日……"——全书情感浓度最高的台词
```

#### 丁仪
```
句长：中短交替，跳跃式思维
口语程度：中
特征：物理学家特有的直接，有时看似跑题其实直指本质
标志："想象另一种结果……"、"为虫子干杯！"
```

#### 申玉菲
```
句长：极短——全书最精简的对白
口语程度：中
标志：重复修辞——"停下来试试。" × 3
```

### 三体游戏角色对白
```
周文王/纣王/墨子等：使用文言白话混杂的措辞
语调：庄严、古奥、有一种史诗朗诵感
对比：与现实中角色的口语形成强烈反差
```

## 三、节奏规则

### 句长规则
- **科学线默认状态**：中短句交替（15-40字）
- **叶文洁回忆段**：句长略长（20-50字），有文学性长句
- **古筝行动段**：句长压缩至10-25字，制造窒息感
- **三体游戏线**：句长中等偏短，但措辞文雅

### 段落呼吸
- **调查推理段**（汪淼追踪线索）：3-5行段落，紧密推进
- **叶文洁回忆段**：5-8行段落，允许留白和沉浸
- **三体游戏段**：4-6行，保持叙述节奏但不失细节
- **古筝行动段**：2-3行，每一段都是一个切割的瞬间

### 场景转换速度
- **现代线↔历史线**切换：通过审讯场景作为桥梁
- **现实↔三体游戏**切换：用"汪淼摘下V装具"作为明确边界
- **major场景**至少持续10-15个节点，避免频繁打断

## 四、视角转换

### 原作视角分析
原作是**第三人称有限视角**——每章锁定一个角色，但不进入角色内心作心理描写。叙事者观察角色的外部行为和对白。与三体2最大的不同：原作拥有4+个可切换视角。

### 游戏视角转换方案

**核心规则**：
- 第三人称小说 → 第二人称游戏
- 每切换一个角色视角，必须执行强烈的身份锚定

**视角切换策略**：

| 视角 | 占比 | 过渡方式 |
|:----|:----:|:---------|
| 汪淼（主线） | ~60% | 默认视角。第一行锚定："你叫汪淼。你是一个纳米材料专家。" |
| 叶文洁（历史线） | ~25% | 通过审讯场景切换。锚定物：温度/时间感——"你叫叶文洁。1967年的冬天冷得不像话。" |
| 史强（行动线） | ~10% | 在古筝行动等场景中切换。锚定物：烟味/地面粗粝感 |
| 其他 | ~5% | 环境叙事/不可操作节点 |

### 信息披露策略（关键）
- **叶文洁线**的"她是ETO统帅"真相：在历史线中不提前暴露——玩家以年轻叶文洁的视角经历一切，直到被审讯时才揭示全貌
- **三体游戏线**的"这不是游戏是真实"：通过碎片化线索让玩家逐步拼出真相

## 五、禁止清单

1. ❌ **直接心理描写**——不写"你感到恐惧"，写外部表现（"手在颤抖"、"冷汗浸透了内衣"）
2. ❌ **"意识到"、"明白"、"发现"的主人公内心独白**
3. ❌ **道德评价**——古筝行动中不写"这是残忍的"，只写切割面像镜面一样光滑
4. ❌ **煽情和催泪**——叶文洁的悲剧靠事实本身说话，不靠叙述段催泪
5. ❌ **感叹号滥用**——全书极少使用感叹号，仅在"不要回答！不要回答！不要回答！"等极少数场景
6. ❌ **三体游戏的"游戏化"解读**——不写"这是一个副本/任务"，保持"这是一个真实的世界"的沉浸感
7. ❌ **选项引用玩家未知信息**
8. ❌ **替三体文明做拟人化内心描写**——三体人是全透明的，不需要也不应该被描写内心

## 六、风格自检清单

- [ ] 科学线是否保持了刘慈欣的冷叙述——只记录现象，不做情感渲染？
- [ ] 叶文洁线中，悲剧是否通过事实而不是叙述段传达？
- [ ] 史强的对白是否足够口语化——与所有其他角色形成反差？
- [ ] 三体游戏线是否保持了寓言式的古朴语气？
- [ ] 古筝行动段是否用技术细节的精确性替代了情绪描写？
- [ ] 视角切换时，第一句是否执行了身份锚定？
