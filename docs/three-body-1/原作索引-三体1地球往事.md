## 二、节奏分析

### 全书节奏曲线

```
现代线M01-M09：急入（自杀名单→台球→倒计时→宇宙闪烁）→ 持续加速
三体游戏G01-G08：在加速中插入，作为世界观节奏缓冲（慢→快交替）
历史线H01-H11：在审讯M20中插入，长段沉浸式回忆（慢）
古筝E01：全书最高速高潮
真相E02-E03：急停，沉重揭示
虫子E04：舒缓→上扬
尾声E05：极慢，余韵
```

**节奏型判定**：波浪+阶梯混合型——急入→荡开（三体游戏往复）→深潜（叶文洁回忆）→急转直上（古筝）→以静默收束。

### 节奏约束
- 三体游戏段（G01-G08）：每个游戏登录都是独立叙事单元，不宜在游戏内部设密集分支
- 叶文洁回忆段（H01-H11）：长沉浸式，宜用单选确认维持参与感
- 古筝行动段（E01）：极快速，无分支，纯叙事推进
- "虫子"段（E04）：节奏舒缓后的升华点，不可设负面选项

## 三、人物层：角色卡

### 汪淼
```
身份：纳米材料院士，应用物理学家
外表：中年，学者气质，戴眼镜
说话方式：精确谨慎，科学家的思维习惯——先分析后结论
弧光：从被动卷入的科学界人士→主动承担调查责任的行动者
决策模式：初期被动，被事件推着走；后期主动选择相信史强
```

### 史强
```
身份：刑警队长，粗犷实战派
外表：五大三粗，一脸横肉，脏兮兮皮夹克，浑身烟味
说话方式：北京口语，粗俗锋利，"他妈的"高频
```

## 六、改编策略要点

### 叙事结构
原作是双线叙事：现代线（汪淼）和历史线（叶文洁/红岸），三体游戏作为第三条暗线。
- **现代线**作为主线，玩家扮演汪淼
- **历史线**通过叶文洁的审讯口述/回忆场景插入
- **三体游戏线**以游戏内登录的形式呈现，玩家以汪淼的身份进入

### 叙事视角
- 主线：第三人称有限视角（汪淼） → 第二人称"你"
- 叶文洁回忆：以叶文洁为第二人称的闪回场景
- 三体游戏：保持汪淼视角，但场景和氛围完全不同

### 核心状态值（val）
- `variableName` 建议：「灼见」或「余烬」
- 含义：对宇宙真相的认知深度。高=接近真相但越接近越绝望，低=安全但无知

### 章节划分建议
1. 科学边界（M01-M03）→ 2. 倒计时（M04-M09）→ 3. 三体游戏·序（G01-G08）
4. 叶文洁（H01-H06）→ 5. 宇宙闪烁（M10-M13）→ 6. 红岸深处（H07-H11）
7. ETO（M14-M20）→ 8. 古筝（E01）→ 9. 真相（E02-E03）→ 10. 虫子（E04-E05）
