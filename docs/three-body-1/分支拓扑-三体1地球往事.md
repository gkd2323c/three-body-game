## 三、分支拓扑图（文字描述）

### 主线（真结局路径——对应原作）

```
汪淼接受卧底→调查科学边界→发现倒计时→停止纳米项目→宇宙闪烁→
进入三体游戏→结识叶文洁→叶文洁回忆历史（插入）→
ETO集会/突袭→古筝行动→智子真相→虫子宣言→尾声
```

### 主要分支线概要

**分支线A：拒绝卷入（第1章 S05分出）**
- 汪淼拒绝加入"科学边界"
- → 常伟思放他离开
- → 汪淼回到正常生活，但倒计时没有停止
- → 时间快进冷叙述："一个月后，纳米项目彻底停摆。半年后，你习惯了没有倒计时的视网膜。一年后，杨冬的墓碑长满了杂草……"
- → **草率结局**：倒计时归零的那一天，什么都没有发生
- 汪淼永远不知道答案

⚠️ **防误触/再试一次机制**：进入 Branch A 草率结局后，在结局界面提供「世界线重组」按钮，允许玩家无惩罚地跳转回 `ch1_wm_meeting`（常伟思邀请节点）重新选择，不丢失已解锁的flag/成就。

**分支线B：调查路径选择（第2章分出）**

- 倒计时出现后，汪淼的选择决定信息获取渠道
- 选申玉菲：直接接触"科学边界"核心，更快接触三体游戏
- 选丁仪：更深入理解物理学危机，但进展较慢
- 选史强：获得警方资源，更早接触ETO

⚠️ **收敛函数设计（强制）**：三条路径无论选择哪条，都会在获得阶段性线索后，通过一个不可绕过的**同步锁**汇合到密云射电基地（M07 宇宙闪烁确认）。

  | 路径 | 锁的形式 |
  |:----:|:---------|
  | 申玉菲线 | 申玉菲主动提出"带你去见一个能证明这一切的人" |
  | 丁仪线 | 丁仪联系沙瑞山，说"观测站有东西要你看" |
  | 史强线 | 史强直接逮捕申玉菲→审问→获得密云基地线索 |

  三条路径的节点数为：申玉菲路径 15-20 节点、丁仪路径 12-15 节点、史强路径 10-12 节点。
  汇合后共用密云基地节点，不再因前期路径不同而分叉。

**分支线C：叶文洁的心理底色（第3-5章）**
- 叶文洁在历史线中的选择不改变事件结局，但改变她的心理底色
- "冷漠防备路线"：叶文洁更冷静、更疏离——现代线审讯时更抗拒
- "挣扎悲伤路线"：叶文洁保留更多人性——现代线审讯时更配合
- 影响现代线中获得信息的速度和完整度

**分支线D：古筝行动（第7章）**
- 这是一个纯叙事段，玩家没有改变事件走向的选择
- 但可以在方案讨论环节表达态度（支持/质疑/沉默）
- 态度选择影响史强和汪淼的关系值
- 不影响古筝本身——它是固定事件

## 五、分支文学大纲（顶层概要）

### 主线：汪淼/叶文洁双线

汪淼线：从被动卷入的科学家，到主动调查的"卧底"，到见证宇宙真相的幸存者。
叶文洁线：从创伤到绝望到反抗到幻灭——一条不可逆转的下行弧线，直到尾声的片刻宁静。

双线在ETO聚集中汇合，在审讯室中完成真相交付，在古筝行动中收束。

### 叶文洁心理底色（分支线C）

| 选择倾向 | 效果 |
|---------|:------|
| 多数选择"冷漠防备" | 叶文洁在审讯中更冷静，更少情感表露 → 汪淼获取信息更困难 |
| 多数选择"挣扎悲伤" | 叶文洁在审讯中更脆弱 → 汪淼获取更完整的信息 |
| 混合选择 | 中立路线，叶文洁的倾诉程度居中 |
