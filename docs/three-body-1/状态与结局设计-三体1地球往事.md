## 一、主状态值 val

### variableName：灼见

**含义：你对宇宙真相的认知深度——以及它灼穿你心灵的深度。**

> "射手"假说：一个神枪手在靶子上每隔十厘米打一个洞。靶子上的二维智能生物发现了一个伟大的定律——"宇宙每隔十厘米，必然会有一个洞。"
> "农场主假说"：火鸡科学家经过一年的观察发现了定律——"每天上午十一点，就有食物降临。"感恩节那天，食物没有降临。

高「灼见」意味着：**看穿了世界的底牌——但底牌是一张冷笑的鬼牌。**
低「灼见」意味着：**活在日常的幻觉中——安全但无知，世界还是稳定的。**

| 范围 | 状态描述 | 体验质感 |
|:----:|:---------|:---------|
| 0-20 | 日常——事物的表面 | 世界正常运转。NPC说的话是字面意思。物品就是物品。 |
| 21-40 | 疑惑——边缘开始模糊 | 开始注意到一些不协调。偶尔怀疑"这真的是巧合吗？" |
| 41-60 | 灼烧——看得见裂缝 | 物理学的幻象开始剥落。日常物品在眼中偶尔呈现分子层面的不确定性。NPC的对话开始被解读出潜台词。 |
| 61-80 | 灼痛——真相逼近 | 世界变得半透明。看到一台电脑时文本可能插入"这些集成电路随时可能变成一锅粥"。史强的玩笑话听起来像黑色预言。 |
| 81-100 | 灼穿——与真实对视 | 一个没有规律的世界。三体文明的真相如同灼热的铁烙在视网膜上。 |

### initialVariable：35

汪淼在故事开始时是一个正常的应用物理学家——对前沿物理学的危机有所耳闻，但没有亲身经历过。35 是「疑惑」区间的上沿，给玩家在早期看到倒计时后快速攀升的空间。

### 关键阈值

| 阈值 | 效果 |
|:----:|:------|
| 30 | 首次看到倒计时后自动突破此阈值 |
| 50 | 宇宙闪烁后——世界不再安全，文本渲染开始呈现"裂缝" |
| 70 | 进入智子真相揭示段——世界彻底变化，文本渲染切换为"灼穿"模式 |
| 90+ | 全书末尾——面对真相后的宁静，反而回落到一种冷彻的清晰 |

### val 变动的设计原则

在《三体1》中，val 的变动主要来自**信息获取**而非态度选择：
- 每获得一条关键信息，灼见自动上升
- 态度选择决定**增幅大小**（追问/沉默/回避）
- 叶文洁历史线的选择不影响她的命运，但影响汪淼视角的灼见增速

## 六、「灼见」文本渲染规则

**这是《三体1》最核心的机制创新——val 不只是数值，它改变玩家看到的世界本身。**

### 低灼见（0-40）——日常世界
```
描述申玉菲的别墅：冷清，充满书卷气。书架上是中英文物理期刊，茶几上放着一杯没喝完的茶。
描述作战中心：大厅里一片纷乱——电脑设备胡乱堆放，电线缠在地上，烟雾像晨雾般在半空浮了一层。
```

### 中灼见（41-70）——裂缝出现
```
描述申玉菲的别墅：冷清，书卷气。但你注意到书架上的物理期刊——最近几期的封面设计很奇怪，标题似乎在用一种你读不懂的方式排列。
描述一台电脑：它看起来正常。但你突然想到——如果物理规律真的不均勻，这台电脑里的硅原子随时可能不再按半导体定律工作。
描述史强的话：他说的每一句粗话，现在听起来都像是一种黑色预言。
```

### 高灼见（71-100）——世界灼穿
```
描述日常物品：你看到一只杯子。你想到它是由原子构成的，原子又是由更小的粒子构成的——那些粒子没有固定的位置，没有确定的动量。也许下一秒，它们就不再愿意保持杯子的形状了。
描述NPC的对白：对面的人在说话。但你觉得——如果宇宙是一块靶子，我们都是靶子上的二维生物。一个神枪手在十厘米外打了一个洞。我们称之为"定律"。
```

### 实现方式
- 在 segments 中使用 condition 控制文本内容：`{ "condition": "val >= 50", "text": "..." }`
- 高灼见时 NPC 的同一句对白可以加上不同的系统旁白注释

### ⚠️ 选项劣化/认知污染（val ≥ 80 时生效）

高灼见不仅改变描述文本——它**劣化选项本身**。当 val ≥ 80 时，选项文本从"可操作的动作"退化为"被真相灼穿后的本能反应"：

| 状态 | 选项示例 |
|:----:|:---------|
| 常规（val < 80） | `"检查纳米测试数据"` |
| 常规（val < 80） | `"给史强打电话询问进展"` |
| 高灼见（val ≥ 80） | `"检查那些数据……如果那些无规律的数字真的能被称为「数据」的话。"` |
| 高灼见（val ≥ 80） | `"给史强打电话。倾听那个凡人毫无意义的喧嚣，假装世界依然稳定。"` |

**设计原则**：高灼见下的选项仍然指向同一个 next 节点，但选项文本本身的"变质"让玩家感受到角色的认知正在被真相腐蚀。这不是惩罚——这是角色扮演的深度。

## 七、叶文洁历史线的宿命论处理

### 核心原则

**历史线中的关键事件不可变**——叶文洁必然会：
1. 目睹父亲被打死
2. 进入红岸基地
3. 向太阳发射信号
4. 收到"不要回答"
5. 回答——暴露地球
6. 杀死雷志成和杨卫宁
7. 参与创立ETO

### 玩家选择的作用域

玩家在历史线中的选择**不改变"发生了什么"，但改变"叶文洁的感受是什么"**：

### ⚠️ ye_psyche × trust_ye 乘法解锁公式

在第5/6章审讯节点中，叶文洁核心供词（`ye_confessed` flag）的解锁条件不是单一阈值，而是**历史线底色的放大器**：

```
if ye_psyche == "cold":
    所需 trust_ye ≥ 8   # 冷酷底色需要极高度信任才能破冰
elif ye_psyche == "warm":
    所需 trust_ye ≥ 2   # 温情底色下叶文洁主动倾诉
else:  # neutral
    所需 trust_ye ≥ 5
```

**设计意图**：让玩家在历史线中做出的感性选择（冷漠/挣扎/混合）在现代线审讯场景中获得明确的机制回馈——这不是无形的数值涨跌，而是直接影响能不能拿到关键供词的硬门槛。

| 选择方向 | 效果 | 对现代线的影响 |
|:---------|:-----|:--------------|
| 冷漠防备 | 叶文洁更早对人类绝望，发射信号时更果断 | 审讯中更封闭——汪淼获得信息更困难 |
| 挣扎悲伤 | 叶文洁每一次选择都伴随内心挣扎 | 审讯中更脆弱——汪淼获得信息更完整 |
| 混合选择 | 中性路线 | 信息获取速度居中 |

### 历史线的互动方式

- 历史线以**闪回/回忆**形式呈现（在审讯室场景中触发）
- 玩家以叶文洁的第二人称经历过去
- 重要转折点（如"是否回答"）只提供态度选项，没有命运选项
- 但态度选项积累的 `ye_psyche` 变量直接决定现代线审讯节点的可获取信息量
