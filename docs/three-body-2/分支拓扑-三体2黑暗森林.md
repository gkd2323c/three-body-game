## 三、分支拓扑图（文字描述）

### 主线（真结局路径——对应原作）

```
序章（作为罗辑经历）→ 被选为面壁者 → 庄园生活 + 庄颜 → 庄颜被带走 → 
咒语发送 → 冬眠 → 危机205年唤醒 → 面壁计划被废除 → 水滴毁灭舰队 → 
章北海叛逃 → 黑暗战役 → 罗辑悟出黑暗森林 → 威慑建立 → 团聚
```

### 主要分支线概要

**分支线A：泰勒线（1.3分出）**
- 起点：泰勒访问日本/阿富汗
- 玩家在泰勒线中的选择影响泰勒是否自杀、量子舰队计划是否被破壁
- 如果泰勒在破壁前成功：量子舰队计划推进（但最终仍然无效，三体不在乎）
- 如果泰勒破壁→自杀：触发罗辑的震撼（泰勒的崩溃加速罗辑的思考）

**分支线B：雷迪亚兹线（1.4分出）**
- 起点：雷迪亚兹访问洛斯阿拉莫斯
- 核心：恐日症是否加深？
- 影响结局：如果雷迪亚兹未患恐日症，他在下部苏醒后可能独立行动

**分支线C：章北海线（多结局）**

- **C1（原作线）**：暗杀成功→劫持自然选择号→黑暗战役中牺牲（"没关系的，都一样"）

- **C2a（分支线·技术断头）**：暗杀后暴露被捕 → **「阴影中的殉道者」Bad End**
  
  ⚠️ **硬性限制（不可绕过）**：
  在现实的军事与法治逻辑下，章北海一旦在公元世纪因谋杀航天科学家被捕，
  将彻底失去进入太空军高层的合法性 → 后续数章章北海线无以为继。
  
  因此 C2a 是**高震撼力的单向剧情陷阱**——
  如果玩家在暗杀中留下证据暴露了自己，游戏在此处以 Bad End 提前收束：
  章北海被审判，其真实动机（逃亡主义）在法庭上被公布，引发社会激荡，
  但玩家已无法继续操控章北海线。
  → 后续章北海线的剩余节点全部坍塌为"不可达"

- **C2b（分支线·硬核政治博弈）**：暗杀后暴露，但**被军方高层内部压下**
  
  触发条件：玩家在前期章北海线中获得了常伟思等高层的信任标志
  
  逻辑补丁：常伟思等军方高层看穿了章北海的真实意图（逃亡主义≠失败主义），
  选择在内部隐瞒真相，动用特权将章北海「战略冷冻」至未来，
  以确保其在末日之战前夕苏醒并能延续下半场戏份。
  
  → 章北海被强制冬眠，跳过中间章节，直接在下部唤醒
  → 代价：跳跃期间失去对主线的影响能力，但保留后续节点可达性

- **C3（分支线）**：黑暗战役中章北海先发制人成功→章北海带领舰队远航

**分支线D：庄颜线的选择**
- 起点：庄颜来到庄园
- 核心：罗辑对庄颜的真诚程度
- 结局影响：庄颜是否理解罗辑、是否自愿离开（影响威慑结局时的团聚方式）

**分支线E：罗辑的黑暗森林提前推导**
- 机会窗口：冰湖落水节点
- ⚠️ **硬性限制（不可绕过）**：
  玩家在冰湖阶段**只能达成「理论雏形」或「直觉闪现」**。
  核心的「咒语验证（恒星被毁灭）」是物理沙盒铁律——
  200年光速延迟 + 宇宙猎手清理恒星的漫长周期 = **绝对不可跳过的历史留白**。
  
  具体而言：
  - 冰湖顿悟的上限：玩家可以推导出「宇宙中存在某种清除机制」「暴露是危险的」
  - 冰湖顿悟的下限：但无法精确到「黑暗森林」的完整模型，更无法确认「咒语一定能摧毁恒星」
  - 验证只能来自冬眠200年后的观测——这段历史进程（大低谷→文艺复兴→舰队傲慢）不可压缩

- **提前推导的代价（强制收束回主线）**：
  如果玩家在冰湖收集了足够多的线索 → 触发 ETO 高强度饱和刺杀 →
  罗辑被迫提前进入强制冬眠 → 回到原作主线的冬眠时间线。
  
  效果：满足玩家智力成就感（"我猜到了"）但不破坏主线时间线的物理沙盒。
  
  val 影响：提前推导成功 → 唤醒时 val 初始值 +15（认知优势），但社会融入更困难（知道真相却无人相信）

## 五、分支文学大纲（顶层概要）

> 详细文学大纲将在第4步（状态系统设计）+ 第5步（分段写作）之间逐条展开。
> 以下仅描述各主要分支线的方向基调。

### 主线：罗辑的真结局路径
- **叙事**：从玩世不恭的知识分子、到被迫承担的面壁者、到以爱为驱动的孤独思考者、到最后手握宇宙公理的威慑者。完整呈现"普通人如何在绝境中成长为神"的弧光。
- **关键节点**：咒语发送→冰湖觉醒→黑暗森林推导→墓地终极对峙
- **val 曲线**：从初始中等值出发，经历大幅波动后，在结局攀升至顶峰

### 分支线A：泰勒的两种结局
- **A1（原作线）**：破壁→自杀→罗辑目睹，加速罗辑的成长
- **A2（分支线）**：泰勒计划被PDC阻止但未破壁→泰勒活到危机205年→在下部与罗辑重逢

### 分支线C：章北海线（双结局）
- **C1（原作线）**：暗杀成功→劫持自然选择号→黑暗战役中牺牲（"没关系的，都一样"）
- **C2（分支线）**：暗杀后暴露→章北海被审判→他的逃亡信念是否影响后续决策
- **C3（分支线）**：黑暗战役中章北海先发制人成功→章北海带领舰队远航

### 分支线D：庄颜的密度
- 根据罗辑在庄园阶段对庄颜的真诚程度，庄颜在末日阶段的行为不同
- 真诚度高→庄颜在威慑建立后主动寻找罗辑
- 真诚度低→庄颜选择留在冬眠中，罗辑独自面对末日

## 六、过渡节点规划

### ⚠️ POV 切换防呆标记（强制——针对第二人称灵魂互换）

本游戏在罗辑、章北海、希恩斯之间频繁平行剪辑。同一个「你」在几个身体之间跳跃，
如果过渡段没有强制锚定，玩家极易产生认知混乱。

**规则：每次视角切换，新章的第一句必须执行感官锚定。**

| 切换方向 | 锚定物 | 示例 |
|:--------:|:-------|:-----|
| → 罗辑线 | 温度/触觉 | `壁炉的余烬噼啪一响，你手指间夹着半杯凉透的红酒。` |
| → 章北海线 | 压力/束缚 | `军服的领扣压迫着你的喉咙，你抬手想松开它。` |
| → 史强 | 烟味/地面粗粝感 | `烟灰掉在你的裤子上，你拍了拍，发现手指沾了灰。` |
| → 希恩斯 | 冷光/白雾/全息星海 | `解析摄像机的冷光嗡嗡响着，你的左太阳穴有点发麻。` |

**铁律**：
1. 切换后前 3 个 segments 必须密集使用该视角的专属锚点
2. 禁止在过渡段用旁白说明"你现在是XXX"——用环境说话
3. 视角切换发生的位置只能是 chapter 边界（不可在章中间切换）

基于第一步事件链中的衔接判定：

| 源事件 | 目标事件 | 衔接类型 | 处理方式 | 过渡方式 |
|--------|---------|---------|---------|---------|

| P07→P08 | 罗辑离去→伊文斯对话 | 空间过渡+时间回溯 | 章节分割，用 scene 切换 | 在新节第一个节点用 scene.arrival + 时间标注 |

| U18→U19 | 早餐→车祸 | 时间过渡+情绪转折 | 需补1个过渡节点 | 独立的"走出餐厅"节点，ambient 变化暗示转折 |

## 七、节奏约束汇总

| 节奏区域 | 位置 | 约束 |
|---------|:----:|------|
| ○慢 | 序章·杨冬墓前对话 | 对话段不应插入多选，保持沉浸；用态度分支保持参与感 |
