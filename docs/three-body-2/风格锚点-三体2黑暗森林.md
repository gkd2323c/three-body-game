# 风格锚点：《三体2：黑暗森林》

## 一、叙述温度

### 基础温度：冷叙述 + 暗涌型

刘慈欣的招牌风格：**写灾难像写天气预报，写悲伤靠外部细节暗示。**

示例对比：
- ❌ 不写："罗辑感到无比绝望"
- ✅ 写："罗辑靠着柱子滑坐下来……他感到周围的一切再次崩塌，化做岩浆自顶而下。"

### 温度变化规则

| 场景类型 | 温度 | 执行指引 |
|---------|------|---------|
| 宇宙/物理现象 | 纯冷叙述 | 镜头拉到最近，只记录现象本身，不做任何情感渲染 |
| 人类集体行为 | 反讽（冷叙述+荒诞内容） | 用平静语气写疯狂场面（如全人类狂欢、裸体派对） |
| 角色独处/思考 | 克制抒情 | 通过外部环境暗示内心（寒冷、星空、火光） |
| 角色对白场景 | 接近日常温感 | 对白本身承载情感，叙述段仅写动作和表情 |
| 高潮/灾难 | 极冷（加速到几乎无叙述） | 1分18秒毁灭100艘战舰——只用数字和事实，不用形容词 |

### 游戏正文指引
- segments 默认用短句、无修饰的叙述
- 极端情况下允许一句感官细节（温度、光线、气味），但绝不出现直接情绪形容词
- 唯一的"温度出口"是对白——角色的情感通过对白本身传递

### ⚠️ 冷叙述的补丁——「环境即反馈」原则（强制）

**问题**：冷叙述在小说中成立，因为读者自行消化情绪。但在互动游戏中，选择后没有情绪回馈，玩家会感到"选了什么都一样"——出现底层动机危机。

**规则：选择之后不要写情绪，但要改变世界。**

| 不要这样做（❌） | 要这样做（✅） |
|----------------|--------------|
| "你感到一阵寒意。" | "空调的温度似乎又低了几度，你看到窗玻璃上结了一层薄霜。" |
| "这是一个残忍的决定。" | "方案通过。屏幕上的红点瞬间熄灭了三分之一，主控室里只剩下散热风扇沉闷的嗡嗡声。" |
| "庄颜看起来很悲伤。" | "她转过头去。壁炉的火光只照亮了她半边脸，另一半隐在阴影里。" |
| "大家都很震惊。" | "没有人说话。墙上的信息屏全黑了，整间屋子只剩下自己的呼吸声。" |

**三层反馈机制（按选择重要程度递进）：**

| 层级 | 选择类型 | 反馈形式 | 示例 |
|:----:|:---------|:---------|:-----|
| 1 | 日常/态度选择 | ambient/theme 微变 | 选"追问"→ ambient=static（电流声渐强）；选"沉默"→ ambient=none（静得耳鸣） |
| 2 | 路径分支 | scene/场景变化 | 选"走这条路"→ 下一节点的场景 description 出现裂痕/阴影 |
| 3 | 命运分支 | progress 跳跃 + 环境崩塌/重构 | 选极端选项→下一屏的文本节奏突变（句长压缩/扩张）、出现的字符颜色暗示情绪 |

**冷叙述 × 游戏反馈的执行检查清单：**
- [ ] 这个选择之后，玩家能否仅通过环境描述就感知到自己的选择"被世界承接了"？
- [ ] 如果删掉所有情绪词后选择结果读起来像什么都没发生 — 说明反馈不够，需要加强环境变化
- [ ] ambient 和 effect 是否与选择的情绪色调相匹配？

## 二、对白规则

### 通用规则
- 书面底色，但不端着：角色对白偏书面化（科学家色彩），但不会像学术论文
- 很少用语气词（哦、啊、嗯），除非刻意表现某种状态（如庄颜的天真）
- 几乎没有方言（史强是唯一的例外——但也不是纯方言，是北京口语的书面化）
- 对白密度高：大段对话是推进剧情和传递信息的主要方式
- 潜台词存在但克制：角色说的和想的不完全一致（尤其章北海），但不做"他嘴上这么说，心里却想"式的叙述

### 角色对白规则

#### 罗辑
```
- 句长：中短句为主，偶尔长篇（分析问题时）
- 口语程度：中（知识分子底色的口语）
- 特征：初期玩世不恭，自嘲多，反问多；后期沉稳，很少废话
- 情绪变化：初期嘲讽→中期深情（对庄颜）→后期冷峻坚毅
- 禁区：不会说出煽情台词，不会喊口号
```

#### 史强
```
- 句长：短句，干脆利落
- 口语程度：高——全书最口语化的角色
- 特征：大量北京口语词汇（"扯淡"、"他妈的"、"咋整"）；喜欢用残酷幽默
- 潜台词：几乎无——想到什么说什么，但粗鄙的话语中常有深刻洞察
- 标志：常用"老弟"称呼罗辑，用"孩子们"称呼这个时代的人
```

#### 章北海
```
- 句长：中等，沉稳，每个字都经过选择
- 口语程度：中低——军人的精确和克制
- 特征：极少废话，内心世界从不外露于语言
- 潜台词：极高——说的是A，想的是B，但表面完全正常
- 标志：与父亲的对话只有三句——"要多想"、"想了以后呢？"、"北海，我只能告诉你那以前要多想"
```

#### 庄颜
```
- 句长：短句，句子结构简单
- 口语程度：高——近乎孩子的坦诚和天真
- 特征：用最直接的话表达最纯粹的感受，没有弯弯绕
- 标志："真暖和，真好"、"知道了在哪儿，世界就变小了"
- 禁区：不会说出复杂的长句，不会使用抽象概念

  ⚠️ 纸片人化解药——「艺术性通感」规则（强制）
  庄颜的天真不源于无知，而源于纯粹的艺术感知视角——"她来到这个世界上，就像垃圾堆里长出了一朵百合花"。
  她语言结构简单（短句、直白），但内容可以呈现惊人的感知精度。

  示例对照：
  ❌ 单纯天真（空洞化）：
    "这画真好看。"
  ✅ 艺术性通感（保留了天真句式 + 精准感知）：
    "这画的蓝色……像瓷器碎掉之前的那种蓝。你很难过，但又觉得它碎得很美。"

  规则要点：
  - 句式保持简单（不写复杂从句）
  - 但比喻来自真实的艺术直觉——用视觉/触觉的通感代替抽象评价
  - 常见触发场景：看画、看风景、看雪、看星空、看火光
  - 不触发场景：涉及抽象政治/战略/科学概念时（她应表现出不感兴趣或接不上话）
```

#### 丁仪
```
- 句长：中短句，跳跃式思维
- 口语程度：中
- 特征：物理学家特有的直白和通透，有时看似跑题其实直指本质
- 标志："傻孩子们，快——跑——啊！"、"毁灭你，与你有何相干？"
```

#### 叶文洁（序章）
```
- 句长：短但密度大，每个字有重量
- 口语程度：中（学者的简练）
- 特征：不解释多于一句，说的每句话都在交付重要信息
- 标志："生存是文明的第一需要。""怕没有机会了……"
```

#### 泰勒 / 雷迪亚兹 / 希恩斯（面壁者）
```
- 泰勒：政治家的精确和激情混杂，口号式语言（"妈妈，我将变成一只萤火虫"）
- 雷迪亚兹：粗犷直接，带南美式热情和自我中心
- 希恩斯：温文尔雅，学者式的客气中藏着深不可测
```

## 三、节奏规则

### 句长规则
- **默认状态**：中短句交替（15-40字），不滥用长句
- **分析/思考段**：句长可延长至40-60字，但不超过70字
- **动作/灾难段**：句长压缩至10-25字，用短句制造急促感
- **金句/判词**：15字以内，留出余韵空间

### 段落呼吸
- **慢节奏段**（罗辑思考/庄园生活）：段落长度可到5-8行，允许留白
- **快节奏段**（水滴攻击/章北海暗杀）：段落2-4行，紧密推进
- **过渡段**：3-5行，保持信息量的同时做情绪转场

### 场景转换速度
- **major场景切换**（如地球→太空）：用 chapterTitle 或 scene.arrival 衔接
- **同一场景内的小转场**（如从室内到室外）：一句话带过，不做停留
- **平行剪辑**（罗辑线↔章北海线）：在章边界处使用，不做突然跳跃

### 具体的 segments 节奏指引
```
慢速段（○标记）：每1-2个segments给一次选择（可以是单选项确认）
中速段（●快）：每2-3个segments给一次选择，选项文字简短
高速段（●●快）：每1-2个segments给选择，选项控制在5字以内，用紧迫短促的文本
停顿段（◐停顿）：允许3-5个segments无选择（结局/独白/真相揭露）
```

## 四、视角转换

### 原作视角分析
原作是**第三人称有限视角**——每章锁定一个角色的视角，但不进入角色内心作心理描写。叙事者观察角色的外部行为（动作、对白、表情），但不解释其内心。

### 游戏视角转换方案

**核心规则：第三人称小说 → 第二人称游戏**
- 原作："罗辑感到……" → 游戏："你感到……"
- 原作："他说……" → 游戏：直接写台词
- 原作叙述角色动作 → 游戏：同句保留，转为"你"的描述

**信息披露的调整（关键）**

原作中，读者通过有限视角知道某个角色的外部行为，但不知道其内心想法。游戏中，玩家同样只能看到自己的外部行为和对白，内心状态通过 choices 的选择来表达。

### ⚠️ 视角防呆协议（强制——针对第二人称身份撕裂）

**问题诊断**：玩家刚作为「你（罗辑）」在卢浮宫和庄颜谈恋爱，下一章突然变成「你（章北海）」用陨石子弹暗杀要员。同一个「你」指向两个身体、两种命运，自我认同会产生严重撕裂。

**三条铁律：**

**铁律一：章边界执行「灵魂附体」锚定**
每一次视角切换，新章的第一句/第一个 segment 必须包含**物理感官锚点 + 身份刚印**。

| 视角 | 锚点示例 |
|:----:|---------|
| 罗辑→章北海 | `"军服的领扣压迫着你的喉咙，你抬手想松开它，指尖触到了硬邦邦的——你穿着太空军的作训服，不是那件松垮的夹克。"` |
| 章北海→罗辑 | `"壁炉的余烬噼啪一响，你发现自己瘫在摇椅里，手指间夹着半杯凉透的红酒。没有军衔，没有口令，只有灰尘在光柱里浮动。"` |

**铁律二：每个视角有专属的内核 sensory signature**
- **罗辑**：触觉锚点——温度（壁炉的火/冰湖的水/酒杯的凉），身体接触（握住庄颜的手）
- **章北海**：压力锚点——领口、肩章、武器后坐力、失重时的束缚感
- **史强**：嗅觉/味觉锚点——烟味、酒气、血腥味

切换后前 3 个 segment 必须密集使用该视角的专属锚点，帮助玩家「重新学习」这个身体。

**铁律三：章北海线的「封闭内心」处理**
- 章北海线的选择**只反映外部行为，不反映内心动机**——玩家选"加入高技术战略研究室"，看不到对话框提示"（你其实在考虑逃亡）"
- 他的真实信念在暗杀事件节点才强制揭示：届时回溯玩家之前的全部选择，用章北海自己的话重新诠释
- 这意味着——章北海线玩家的选择可能和最终揭示的动机相反，这正是原作体验

**多视角切换方案**
- 主线以罗辑为第一视角（约 70% 节点）
- 章北海线以章节切换方式嵌入（约 20-25% 节点）
- 张援朝/普通人类线作为环境叙事（非可操作节点，通过氛围/新闻片段嵌入）

## 五、禁止清单

在本作的游戏改编中，**绝对不能出现**：

### 叙述禁区
1. ❌ **直接心理描写**——不写"你感到悲伤/愤怒/绝望"。用外部表现替代（"你的手在颤抖"、"你发现自己的牙关紧咬"）
2. ❌ **"意识到"、"发现"、"明白"**等认知动词的主人公内心——不写"你意识到自己被骗了"，写对话中的揭示性信息
3. ❌ **道德评价**——不写"这是对的/错的/残忍的/勇敢的"。让事件本身说话
4. ❌ **煽情和催泪**——不通过叙述段直接刺激情绪。情绪通过对白和 choices 的后果自然产生
5. ❌ **感叹号泛滥**——全书几乎不用感叹号，游戏正文中同样严格限制。只在极度的命令/喊叫场景使用
6. ❌ **过度修辞**——不使用连串比喻、拟人、排比。一个精准的意象比一串修辞更有力
7. ❌ **解释主题**——不写"这就是黑暗森林的残酷法则"，不做主题总结。主题通过 story 本身传达

### 对白禁区
8. ❌ **角色说出自己无法知道的信息**——罗辑在早期不能说"黑暗森林"这个词
9. ❌ **角色替作者说出主题**——不写"人类真是个奇怪的物种"这种旁白式对白
10. ❌ **史强的过度文明化**——他不能说"这体现了文明的本质"这种书面对白
11. ❌ **庄颜的过度复杂化**——庄颜的用词和思考深度不能超过一个热爱艺术的年轻女孩

### 选项禁区
12. ❌ **选项中出现玩家此刻未知的信息**——与 step5-writing.md 的铁律一致
13. ❌ **选项替玩家做心情描述**——不写"你感到好奇，问……"，直接写台词"……？"
