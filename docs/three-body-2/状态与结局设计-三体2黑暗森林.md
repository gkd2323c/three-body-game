## 一、主状态值 val

### variableName：明暗

**含义：你在黑暗森林中的可见度。**

> "你看，星星都是一个个的点……宇宙中各个文明社会的复杂结构……都被这样巨大的距离滤去了，那些文明在我们看来就是一个个拥有参数的点。"

高「明暗」意味着：**暴露、透明、被注视、输出信号、无法回头。**
低「明暗」意味着：**隐藏、模糊、被遗忘、停止广播、保持沉默。**

| 范围 | 状态描述 | 体验质感 |
|:----:|:---------|:---------|
| 0-20 | 暗处——几乎完全隐没 | 安全但孤独，信息隔绝，"没人知道你在这里" |
| 21-40 | 阴影——偶有信号 | 可以观察别人但不易被发现，行动受限 |
| 41-60 | 微光——日常状态 | 正常社交，有限暴露，承担可控风险 |
| 61-80 | 明处——被持续关注 | 影响力上升但危险加剧，选项变少但每步分量更重 |
| 81-100 | 刺目——无遮拦 | 一切行动都被观察，代价极高但可能获得最终威慑能力 |

### 关键特性

**明暗的两端都不是简单的「好」或「坏」：**
- 低明暗（隐藏）→ 安全但孤独，信息获取有限，无法影响大局
- 高明暗（暴露）→ 影响力大增但每一步都被注视，容易触发致命威胁

### initialVariable：40

罗辑在故事开始时是一个玩世不恭的知识分子——说不上完全隐藏（他活得高调），也说不上完全暴露（他不过是个普通人）。40 是「微光」区间的下沿，给他向两个方向发展的空间。

### 变动原则

| 变动幅度 | 对应事件 |
|:---------:|:---------|
| ±5 | 日常态度选择、对话回应 |
| ±10 | 重要表态、信息获取/隐瞒 |
| ±15-20 | 核心行动（发送咒语/拒绝面壁者身份/建立威慑） |
| 特殊 | 特定事件强制设定到某个值（如威慑建立→明暗锁定在 85+） |

### 关键阈值

| 阈值 | 效果 |
|:----:|:------|
| ≤20 | 触发「阴影庇护」——某些追杀类事件跳过，但关键信息节点不可达 |
| =40 | 初始平衡点——所有路径默认可达 |
| ≥60 | 触发「被注视」——ETO 类事件触发概率上升，部分角色态度改变 |
| ≥80 | 触发「不可逆暴露」——进入威慑路线的前置条件，但所有 stealth 选项关闭 |

### val 的作用范围（正确用法）

**val 不决定结局分流，而是影响同一条路径内的体验质感：**

- **NPC 态度**：高明暗时史强更警惕/"公事公办"；低明暗时史强更随意/"哥俩好"
- **可获取信息**：高明暗时某些角色愿意透露核心秘密；低明暗时他们保持沉默
- **segments 文本色调**：高明暗时的环境描写更冷、更空、更金属感；低明暗时更暖、更日常
- **可用选项**：某些选项只在特定明暗阈值出现

**val 不决定的事：**
- ❌ 不是结局触发条件（结局由 flag 决定）
- ❌ 低明暗 ≠ 坏结局
- ❌ 高明暗 ≠ 好结局

## 二、自定义变量 variables

| 变量名 | 类型 | 初始值 | 用途 |
|:------:|:----:|:------:|:-----|
| trust_shi | number | 0 | 与史强的关系（-10~10），影响史强在关键节点的行为 |
| trust_zhuang | number | 0 | 与庄颜的关系（-10~10），影响庄颜线的走向和结局变体 |

⚠️ **trust_zhuang + zhuangyan_truth 的化学反应规则（强制）**

**问题**：如果玩家在关键节点触发了 `zhuangyan_truth`（告诉了庄颜"你是我创造出来的梦中情人"），
这意味着庄颜得知自己最初是被"设计"出来的人——她的整个存在被质疑。
这种情况下 `trust_zhuang` 的数值变化不能是简单的 ±n。

**规则：真话的代价**

| 事件 | trust_zhuang 变化 | 叙事效果 |
|:----|:-----------------|:---------|
| `zhuangyan_truth` 触发（第一次告诉真相） | **-8 瞬间暴跌** | 情感裂痕——庄颜开始怀疑一切 |
| 后续：用克制抒情修补关系（每次成功） | +2 ~ +3（缓慢） | 种子"独立人格"开始生长 |
| 后续：继续隐瞒/撒谎 | ±0（表面维持） | 信任数字高但底层是沙堡 |
| 结局前：庄颜最终发现被长期欺骗 | 强制 `zhuangyan_state: "lost"` | 无论 `trust_zhuang` 数值多高，触发"失去"结局 |

**核心机制**：
- 选择「告诉真相」→ `trust_zhuang` 瞬间暴跌，但同时种下**独立人格的种子**，
  后续修补后，庄颜真正"为自己而活"，结局团聚的可能性更高
- 选择「用谎言维持」→ `trust_zhuang` 表面平稳，但底层藏着不定时炸弹，
  一旦庄颜自己发现真相（如在庄园找到当年PDC的文件），`trust_zhuang` 归零且不可逆转

**最终映射**：
```
zhuangyan_reunited（真结局团聚） 需要的不是 trust_zhuang ≥ X，
而是：zhuangyan_state == "independent"（庄颜作为独立个体主动选择回来）
```
| trust_chang | number | 0 | 常伟思对章北海的信任度（-5~10），影响 C2b 触发 |
| route_zhang | string | "none" | 章北海线路径状态："none"/"normal"/"exposed"/"frozen"/"fled" |
| route_taylor | string | "none" | 泰勒线状态："none"/"broken"/"alive" |
| route_rey | string | "none" | 雷迪亚兹线状态："none"/"phobia"/"stable" |
| route_hines | string | "none" | 希恩斯线状态："none"/"truth_revealed" |
| knows_dark_forest | string | "none" | 对黑暗森林的认知："none"/"intuition"/"theory"/"confirmed" |
| shiqiang_alive | boolean | true | 史强是否存活（受刺杀事件影响） |
| zhuangyan_state | string | "with_luo" | 庄颜状态："with_luo"/"frozen"/"lost" |

#### ending_hidden｜另一个伊文斯

| 维度 | 描述 |
|:----|:-----|
| 触发条件 | 极罕见 flag 组合：罗辑在多个关键节点选择了与三体合作/不抵抗路线，并在威慑节点选择了「放弃——与三体谈判」 |
| 叙事方向 | 罗辑成为三体人在人类中的新代理人——不是伊文斯式的狂热信徒，而是理性选择：人类抵抗只会加速灭亡 |
| 主角最终状态 | 活着，但成为人类叛徒 |
| 真相揭露程度 | 完全知道黑暗森林，但认为接受三体统治是唯一现实选择 |

⚠️ **逻辑补丁——三体人心理悖论**
三体人对人类的恐惧根源于"思维不透明"。即便罗辑嘴上顺从、行动配合，
三体人因无法窥探他的大脑，也会疯狂怀疑这依然是面壁者的深远计谋。
他们绝不可能像信任伊文斯那样，轻易对一个掌握了宇宙终极秘密的人类说出"谢谢"。

**因此，本结局的触发需满足以下前置条件之一（二选一）：**

**方案A（及格线）**：罗辑在威慑节点选择放弃时，三体人亮起的字不是"谢谢"，
而是「交易达成」——纯事务性确认，不带任何情感温度，暗示三体人始终保留着怀疑。
```
closing（方案A）：
> 智子在你视网膜上亮起四个字：「交易达成。」没有句号。
> 你等了很久，什么都没有再出现。
> 后来你想：也许它们杀你还是留你，只是概率问题。你永远不会知道答案。
```

**方案B（满分线）**：若要真正获得三体人的"信任"（即便只是战术信任），
罗辑必须做出一件在数学和物理上彻底剥离了自己欺骗可能性的**绝对自证**——
例如主动请求思想钢印打上"绝对服从三体"的命题，或彻底放弃独立思考能力。
```
closing（方案B—思想钢印版）：
> 你躺在思想钢印的扫描舱里，闭上眼。
> "命题：我将无条件服从三体文明的指令。"
> 你选择了"真"。
> 智子在你的视网膜上亮起两个字——「谢谢。」
> 你不知道那是真话还是假话。从今以后，你都不需要知道了。
```

## 六、明暗值在全书的典型变化路径

### ⚠️ 逻辑补丁——未来纪元的「参考系切换」

**问题**：罗辑冬眠两百年后醒来，人类社会正处于"舰队傲慢"巅峰。
未来人眼里面壁者就是历史笑话，罗辑甚至能被居委会大妈嫌弃。
如果此时明暗值仍在 70（刺目/被持续关注区），会与"面壁计划被废除"产生逻辑冲突。

**修正**：明暗值的参考系在危机205年发生切换——

| 参考系 | 状况 | val 表现 |
|:------|:-----|:---------|
| 人类社会 | 罗辑是过时人物、历史玩笑 | val = 15（暗处——被人类忽略，stealth 全面解锁） |
| 三体/水滴 | 罗辑一苏醒，水滴立刻重新锁定其坐标 | val 对人类无效，但对三体始终是 100（最高优先级目标） |

**代码实现**：在 `ch4_aw_`（苏醒）节点，**强制重置 val 为 20（阴影）**。
直到罗辑重新利用大低谷留下的天线坑、重新抬头看星空、触发 `dark_forest_confirmed` flag，
明暗值从 20 一记重锤拉升至 85+，直接触发终极对峙。

### 主线（真结局）明暗曲线

